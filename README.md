# Retail--Sales-Analysis
# 📊 Retail Sales Analysis Project

## 🚀 Project Overview
This project is an end-to-end Retail Sales Analysis using **Python, Excel, Matplotlib, and Power BI**.  
It focuses on analyzing sales performance across categories, regions, and payment methods to generate business insights.

---

## 🛠️ Tools & Technologies Used
- Python (Pandas, Matplotlib)
- Excel
- Power BI
- Jupyter Notebook

---

## 📁 Project Workflow

### 1️⃣ Data Collection
- Created raw sales dataset in Excel (`sales_data.xlsx`)

### 2️⃣ Data Processing (Python)
- Loaded dataset using Pandas
- Created new feature: `Total_Sales = Quantity × Price`
- Performed data cleaning and transformation

### 3️⃣ Data Analysis
- Category-wise sales analysis
- Region-wise sales analysis
- Payment method analysis

### 4️⃣ Data Visualization (Matplotlib)
- Bar chart for Category sales
- Bar chart for Region sales
- Pie chart for Payment distribution

### 5️⃣ Dashboard (Power BI)
- Imported cleaned dataset into Power BI
- Created interactive dashboard with:
  - Total Sales KPI
  - Category-wise visualization
  - Region-wise analysis
  - Payment method distribution

---

## 📊 Key Insights
- Electronics category generated highest revenue
- Card payments contributed the largest share
- Region-wise sales variation identified across dataset
